# AIS приемник на базе AX5043
## Задачи
1. Научиться работать с радиочипом AX5043 при помощи ИИ-инструментов
2. Разработать и протестировать схему ВЧ части AX5043 для приема AIS сигнала
3. Разработать ПО для работы в связке ESP32 и AX5043
4. Провести тестирование AIS приемника, измерить его чувствительность, энергопотребление


## Железо
1. Плата [ESP32-S3-Zero](https://cnx-software.ru/2024/08/07/waveshare-esp32-s3-zero-eto-miniatyurnyj-modul-wifi-i-ble-iot-s-portom-usb-c-do-32-gpio/)
2. Радиочип AX5043 с самодельной обвязкой: TXCO, LDO, ВЧ-часть и тп

## Вводная
После многочисленных попыток "завести" плату E32 170T30D для приема AIS сигнала (получилось, но недостаточная чувствительность на уровне -85 дБм в пакетном режиме) решил попробовать другие варианты. Чип AX5043 по описанию подходит замечательно: умеет работать с кастомной преамбулой, может производить дифференциальное кодирование/декодирование (считай тоже NRZI), умеет искать именно HDLC пакеты и проверять нужную CRC-16-CCITT/X-25. А еще может выдавать IQ сигнал, может работать на частотах до 27 МГц, может выдавать аналоговый сигнал (FM-приемник) и тп. Единственное, данный чип снят с производства и больше не поддерживается производителем. Например, так и не получилось найти и скачать фирменную утилиту AX RadioLab.
